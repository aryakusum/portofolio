# Codebase Stack

## Core Technologies

- **Runtime:** Node.js (via Vite)
- **Language:** JavaScript (ES6+), JSX
- **Framework:** React 19.1.1
- **Build Tool:** Vite 7.1.2

## Styling & Animations

- **CSS Framework:** Vanilla CSS, TailwindCSS 4
- **Animation Libraries:**
  - Framer Motion
  - GSAP
  - Anime.js
  - AOS (Animate on Scroll)
  - Animate.css
- **Scroll Effects:** Lenis (Smooth Scrolling)
- **3D Graphics:**
  - Three.js
  - @react-three/fiber
  - @react-three/drei
  - @react-three/rapier (Physics)
  - OGL (WebGL framework)

## Tools & Configuration

- **Linter:** ESLint (eslint.config.js)
- **Deployment:** gh-pages
- **Icons:** lucide-react, react-icons, remixicon
- **Typography:** @fontsource/inter, @fontsource/playfair-display
